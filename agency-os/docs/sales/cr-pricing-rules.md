# CR Pricing Rules

## 目的
- 將變更需求標準化估價，避免超範圍免費工作。

## 流程
1. 收到需求並文字化。
2. 判定是否超出 SOW。
3. 分析時程、成本、風險。
4. 產出 CR 報價與交期。
5. 客戶簽核後排程執行。

## 估價模型
- `CR 價格 = 基礎工時成本 + 風險係數 + 時程壓縮係數 + 管理成本`

## 規則
- 未簽核 CR 不排程。
- 「先做再談」原則拒絕。
- 每筆 CR 必須更新專案 `01_SCOPE_AND_CR.md`。
- 範圍判定與邊界依 `docs/operations/scope-change-policy.md`。
