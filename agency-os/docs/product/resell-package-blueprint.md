# Resell Package Blueprint

## 目的
- 讓 Agency OS 可以作為可銷售產品（顧問導入 + 系統套件）。

## 商品結構
- Core System License（文件、流程、腳本）
- Onboarding Service（導入與客製）
- Operations Service（持續維運與優化）

## 交付包內容
- 治理文件（docs）
- 自動化腳本（sync/health/guard/scheduling）
- tenant 模板（可複製）
- project-kit 模板（可開案）
- 驗證報告（health + closeout）

## 產品分級（建議）
- Standard：單公司/單區域導入
- Pro：多公司/多區域導入 + 自動化排程
- Enterprise：合規加強、稽核證據包、跨時區運營

## 商務模式（建議）
- 初始導入費 + 月維運費
- 客製功能 CR 收費
- 年度升級與審核服務

## 品質保證邊界
- 可承諾流程完整性與可追溯性
- 不承諾「零風險」，承諾「可觀測、可回滾、可復原」
